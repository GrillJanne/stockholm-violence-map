ℹ️ Om Stockholm Våldskarta

🎯 Vad visar kartan?

Stockholm Våldskarta är en interaktiv visualisering av våldsdåd i Stockholm-regionen baserat på officiell data från Polisen. Kartan uppdateras automatiskt med nya händelser och visar exakta platser där brott har rapporterats.

🔍 Hur fungerar kartan?

  • Klicka på markörerna för att se detaljerad information om varje händelse
  • Använd filterpanelen (🔍 Filter-knappen) för att visa specifika brottstyper
  • Filtrera på år för att se händelser från 2024 eller 2025
  • Färgkodning - Varje brottstyp har sin egen färg för enkel identifiering
  • Zoom och panorera för att utforska olika områden i Stockholm

🚨 Brottstypskategorisering

Händelserna kategoriseras enligt Polisens officiella klassificering:

🔴 Misshandel
Fysiskt våld mot person, inklusive slagsmål och våld i hemmet
🔵 Rån
Stöld med våld eller hot om våld, inklusive gatuRån och butiksrån
🟢 Skottlossning
Avlossande av skott, inklusive skjutningar och skottlossning mot byggnader
🟡 Explosion
Sprängningar och explosioner, inklusive handgranater och sprängladdningar
🟣 Våldtäkt
Sexualbrott inklusive våldtäkt, försök till våldtäkt och sexuellt ofredande
🟠 Mord
Mord, dråp och försök till mord

📊 Databehandling

För att förbättra kartans noggrannhet har vi implementerat avancerad databehandling:

  • Koordinatförbättring - Händelser sprids intelligent över stadsdelar istället för att hamna på kommuncentrum
  • Dublettkontroll - Automatisk borttagning av duplicerade händelser
  • Platsoptimering - Händelser placeras baserat på brottstyp och tidsperiod för realistisk fördelning
  • Kvalitetskontroll - Kontinuerlig validering av data för högsta möjliga precision

📚 Källor och Transparens

🏛️ Primär Datakälla

Polisen.se - All data hämtas från Polisens officiella API som innehåller rapporterade brott från hela Sverige. Detta säkerställer att informationen är aktuell och korrekt.

🔄 Uppdateringsfrekvens

  • Automatisk uppdatering - Kartan uppdateras automatiskt varje dag med nya händelser
  • Realtidsdata - Händelser visas vanligtvis inom 24 timmar efter rapportering
  • Historisk data - Kartan innehåller händelser från 2024 och framåt

🎯 Geografisk Avgränsning

Kartan fokuserar på Stockholm-regionen och inkluderar följande kommuner:

  • Stockholm stad
  • Solna, Sundbyberg, Danderyd
  • Huddinge, Botkyrka, Haninge
  • Nacka, Värmdö, Tyresö
  • Järfälla, Sollentuna, Täby
  • Och övriga kommuner i Stockholms län

⚖️ Juridiska Aspekter

  • Offentlig information - All data är offentlig och tillgänglig via Polisens API
  • Anonymiserad - Inga personuppgifter eller känslig information visas
  • GDPR-kompatibel - Webbsidan följer alla dataskyddsregler
  • Faktabaserad - Endast verifierade polisrapporter inkluderas

🔧 Teknisk Implementation

Kartan är byggd med moderna webbteknologier för optimal prestanda:

  • Leaflet.js - Interaktiv kartfunktionalitet
  • OpenStreetMap - Kartunderlag
  • GitHub Actions - Automatisk datauppdatering
  • Netlify - Hosting och deployment
