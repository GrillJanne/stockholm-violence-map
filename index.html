ℹ️ Om Kartan

Denna interaktiva karta visar våldsdåd i Stockholm-regionen baserat på officiell data från polisen.se. Kartan uppdateras automatiskt med ny information och ger en översikt över brottsutvecklingen i olika områden.

🔍 Funktioner

  • Separata markörer: Varje brott visas som en egen klickbar punkt
  • Smart spridning: Händelser på samma plats sprids ut automatiskt
  • Zoombar karta: Zooma in för att se detaljer och separata händelser
  • Färgkodning: Olika brottstyper har olika färger för enkel identifiering
  • Filteralternativ: Filtrera efter år och brottstyp
  • Direktlänkar: Läs mer om varje brott på polisen.se
  • Responsiv design: Fungerar på både dator och mobil

📊 Datakällor

  • Polisen.se officiella händelserapporter
  • Automatisk uppdatering var 5:e minut
  • Geografisk positionering baserad på platsnamn i rapporter

🔍 Källor och Metodik

All data kommer från polisen.se:s officiella API och händelserapporter. Kartan visar endast våldsdåd som klassificeras som misshandel, rån, skottlossning, explosion, våldtäkt, sexualbrott, mord och olaga hot.

📈 Databehandling

  • Automatisk hämtning från polisen.se API
  • Filtrering på våldsdåd enligt ovan kategorier
  • Geografisk positionering baserad på platsnamn
  • Dublettkontroll för att undvika duplicerad data
  • Automatisk uppdatering för aktuell information

⚠️ Viktiga noteringar

  • Kartan visar endast anmälda brott som publicerats av polisen
  • Mörkertalet kan vara betydande för vissa brottstyper
  • Geografisk precision kan variera beroende på rapportens detaljnivå
  • Data uppdateras kontinuerligt men kan ha viss fördröjning
