ℹ️ Om Kartan

🎯 Vad visar kartan?

Stockholm Våldskarta visualiserar våldsdåd i Stockholm-regionen baserat på officiell data från polisen.se. Kartan uppdateras automatiskt och visar händelser som misshandel, rån, skottlossning, explosioner och andra våldsbrott.

🔄 Hur fungerar det?

Data hämtas automatiskt från polisens öppna API och bearbetas för att ge bästa möjliga platsangivelse. Varje händelse visas som en färgkodad markör på kartan där färgen indikerar brottstyp.

🎨 Färgkodning

Olika brottstyper visas med olika färger för enkel identifiering. Se färgkodningen ovan kartan för detaljer om vilken färg som representerar vilken brottstyp.

🔍 Filter och Navigation

Använd filterpanelen för att:

  • Filtrera på specifika brottstyper
  • Välja tidsperiod (2024, 2025 eller alla år)
  • Anpassa visningen efter dina intressen

🔗 Länka till Polisens Rapporter

Klicka på "Läs mer på polisen.se" i popup-fönstret för att komma till polisens ursprungliga rapport med fullständig information om händelsen.

🔍 Källor och Transparens

📊 Datakällor

All data kommer från polisen.se:s officiella API som tillhandahåller information om anmälda brott. Detta säkerställer att informationen är korrekt och aktuell.

🔄 Uppdateringsfrekvens

Kartan uppdateras automatiskt flera gånger per dag för att visa de senaste händelserna. Nya brott visas vanligtvis inom 24 timmar efter att de rapporterats.

📍 Platsangivelse

Koordinater förbättras intelligent baserat på händelsebeskrivningar och kända områden för att ge så exakt placering som möjligt inom ramen för polisens dataskydd.

🛡️ Integritet och Säkerhet

Inga personuppgifter visas. All information är redan offentlig genom polisen.se och presenteras endast i aggregerad, anonymiserad form.

⚖️ Juridiska Aspekter

Kartan visar anmälda brott, inte dömda. Alla personer är oskyldiga tills motsatsen bevisats i domstol. Data används i enlighet med polisens användarvillkor.
